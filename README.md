## 📖 About the Project

This web tool is a lightweight, single-file interactive application designed to help readers master the concepts found in **"Predators of the Soul"**. Utilizing the metaphors of African wildlife—the Lion of Fear, the Leopard of Anger, and the Hyena of Greed—this game reinforces the spiritual strategies required to live in the **Victory of the Cross**.

Built entirely in a single HTML file, it requires no installation, no database, and works on any modern device.

## 🛡️ The 8 Soul Predators

Face each predator in a themed expedition. Each chapter includes a summary, a knowledge check, and a strategy plan:

| Predator | Spiritual Threat | Strategy |
| :--- | :--- | :--- |
| **Lion** | Fear & Paralysis | The Lion-Killer Action Plan |
| **Leopard** | Human Anger & Wrath | Walking in the Spirit |
| **Wild Dog** | Lies & Speculation | The "Horns" & "Hoofs" of the Word |
| **Buffalo** | Isolation | Staying with the Herd |
| **Tiger** | Sexual Immorality | The Walls of Covenant |
| **Crocodile** | Despair & Depression | The Anchor of Hope |
| **Hyena** | The Love of Money | Stewardship & Simplicity |
| **Cheetah** | Pride & Competition | The Truth of Grace |

## ✨ Features

- **🏆 Token System:** Earn tokens by completing challenges to unlock subsequent chapters.
- **🗡️ Sword Drills:** Interactive memory verse challenges to sharpen your use of the Word.
- **🗺️ Strategy Plans:** Drag-and-drop ordering to prioritize combat steps against sin.
- **📱 Responsive Design:** Full mobile-friendly jungle-themed UI with dark mode.
- **💾 Local Progress:** Auto-saves your expedition status using browser LocalStorage.
